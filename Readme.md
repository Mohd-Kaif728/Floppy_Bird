A fully functional **Flappy Bird** clone built with Python and Pygame. Navigate the bird through pipes, test your reflexes, and aim for the highest score!


## ✨ Features

- 🎯 **Smooth gameplay** with realistic physics
- 🔊 **Sound effects** for wing flaps, scoring, and collisions
- 📊 **Live score tracking** during gameplay
- 💀 **Game over screen** with restart option
- 🎨 **Retro-style pixel graphics** with animated bird
- ⚡ **Increasing difficulty** as you progress

## 🎮 How to Play

The goal is simple: **fly the bird through the gaps between pipes** without hitting them. Each pipe you pass gives you 1 point. The game ends if you hit a pipe or the ground.

### Controls

| Action | Key |
|--------|-----|
| **Jump/Fly** | `Space` or `↑ (Up Arrow)` |
| **Restart** | `Enter` (after game over) |
| **Quit Game** | Close the window |

## 📋 Prerequisites

Before running this game, make sure you have:

- **Python 3.6 or higher** installed
- **Pygame library** (version 2.0+)
